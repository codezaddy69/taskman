# TaskMan 📅

AI-powered calendar and todo helper that prepares you for everything.

## Vision

Never start a meeting or task unprepared again.

## Tech Stack

- **Frontend:** Next.js (App Router) + TypeScript + shadcn/ui + Tailwind CSS
- **Backend:** Express.js API + Socket.io
- **Database:** PostgreSQL (Supabase) + Prisma ORM
- **AI:** Provider-agnostic (BYOAI architecture)
