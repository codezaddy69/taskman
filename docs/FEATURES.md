# TaskMan - AI Calendar & Todo Helper
## Core Concept: The Preparation Loop

**TaskMan isn't just a storage app — it's your preparation assistant.**

1. **Add** something to your calendar or todo list (quick, minimal input)
2. **TaskMan researches** it — who, what, where, when, why
3. **TaskMan prepares** you with 3 key things you need to know/do
4. **TaskMan schedules** todos into optimal calendar slots
5. **You execute** → earn XP → level up → get better at life

---

## Core Flow

### 1. Calendar Items → AI Preparation

**You add:**
```
"Meeting with Sarah from Design tomorrow at 3pm about the Q2 roadmap"
```

**TaskMan does:**
- Looks up who Sarah is (your contacts, previous meetings, Slack, LinkedIn if connected)
- Understands "Q2 roadmap" context (recent docs, related tasks, project history)
- Finds "Design" team context (who's involved, recent projects, pain points)
- Prepares 3 key preparations:

**Preparation Card Shows:**
```
📅 Meeting with Sarah (Design Lead)
   Tomorrow, 3:00-4:00pm

🎯 WHAT YOU NEED TO KNOW

1️⃣ Sarah's priorities this quarter
   → Just shipped the onboarding redesign
   → Pushing for better mobile UX
   → Concerned about design debt in legacy features

2️⃣ Recent context
   → You reviewed wireframes last Friday
   → Design team is 2 days behind schedule
   → Stakeholders want fewer meetings, more async

3️⃣ What to bring
   → Updated roadmap mockups
   → Analytics on new signup flow
   → Questions about timeline constraints

📎 ATTACHMENTS
   - Q1 Design Review.pdf
   - Roadmap v2.3 (shared last week)
   - Sarah's design priorities doc

💡 CONVERSATION STARTERS
   "How's the mobile testing going?"
   "What's blocking you most right now?"
```

**Sources it pulls from:**
- Your calendar (previous meetings with Sarah)
- Your tasks (related design tasks)
- Email (recent threads)
- Slack (if integrated)
- Documents (Google Drive, Notion, etc.)
- Contacts (profile info, notes)
- Web search (public info, company updates)

---

### 2. Todo Items → AI Enrichment + Calendar Scheduling

**You add:**
```
"Call mom about birthday plans"
```

**TaskMan does:**

**Enrichment Card:**
```
📝 Call mom about birthday plans

🎯 CONTEXT
   → Mom's birthday: March 12th (in 2 weeks)
   → Last year: dinner at Italian place
   → She mentioned wanting to try hiking this year
   → Gift ideas from conversation: hiking boots, spa day, family photo session

⏰ TIME ESTIMATE
   30-45 minutes (calls with her usually run long)

📍 LOCATION SUGGESTIONS
   If planning something physical:
   → Local hiking trails: Mt. Tam, Lands End, Presidio
   → Weather: rain expected next weekend, plan indoor backup
```

**Calendar Suggestions:**
```
📅 SCHEDULE THIS TODO

Best slots (based on your patterns, energy, and her timezone):

⭐ RECOMMENDED
   Thursday 6:00pm
   → You're free after work
   → She's usually relaxed evenings
   → Low conflict day for you

   [SCHEDULE]  [MOVE]  [IGNORE]

⚡ QUICK SLOT
   Tomorrow 7:30am
   → You wake up early Wednesdays
   → Good for quick decisions
   → Before your 9am meeting

   [SCHEDULE]  [MOVE]  [IGNORE]

🗓️ LATER THIS WEEK
   Friday 12:30pm (lunch break)
   Saturday 10:00am (morning person)

   [SCHEDULE]  [MOVE]  [IGNORE]
```

**Once you schedule:**
- Todo becomes a calendar event
- Prep card shows up on the event
- You can add attachments, notes, sub-tasks

### How XP Works

**Base XP:**
- Quick task (<15 min): 5 XP
- Normal task (15-60 min): 10 XP
- Long task (1-3 hours): 25 XP
- Major task (3+ hours): 50 XP

**Multipliers:**
- Completed on time: ×1.5
- High priority task: ×2.0
- First task of the day: ×1.3
- Streak bonus (7+ days): ×2.0

**Penalties:**
- Overdue task: -5 XP
- Rescheduled 3+ times: -2 XP

### Level System

**Level 1-10: Novice**
- Unlock basic analytics
- New avatar frames
- Small badge rewards

**Level 11-25: Apprentice**
