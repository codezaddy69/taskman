# User Personas

Understanding who we're building for helps prioritize features and design decisions.

## Primary Personas

### 1. The Overwhelmed Professional
**Name**: Alex
**Age**: 32
**Occupation**: Product Manager at a tech company
**Tech Savvy**: High

**Pain Points:**
- 50+ meetings per week
- Constant context switching
- Misses deadlines despite working long hours
- Email overload
- Feels "busy but not productive"
- Calendar looks like Tetris

**Goals:**
- Regain control of schedule
- Focus on deep work
- Reduce meeting load
- Actually ship work, not just talk about it
- Work sane hours

**What They Need:**
- Meeting conflict resolution
- Suggested time blocks for deep work
- Task prioritization that actually works
- Quick capture (don't lose ideas in the chaos)
- Daily plan that feels achievable

**Will Pay For:** Yes, subscription is fine if it saves time

---

### 2. The Freelancer / Solopreneur
**Name**: Jordan
**Age**: 28
**Occupation**: UX Designer and consultant
**Tech Savvy**: Medium-High

**Pain Points:**
- Multiple clients = multiple calendars and deadlines
- Inconsistent income makes every deadline critical
- No one to remind them of things
- Work bleeds into personal time
- Forgetting to send invoices
- Tax season horror

**Goals:**
- Never miss a client deadline
- Separate work and personal life
- Get paid on time
- Know when to say "no" to new work
- Feel in control

**What They Need:**
- Multiple calendar support
- Deadline alerts (aggressive, please)
- Income tracking tied to tasks
- "Business hours" enforcement
- Client-specific task lists

**Will Pay For:** Yes, but cost-sensitive. Needs clear ROI.

---

### 3. The Student with ADHD
**Name**: Sam
**Age**: 21
**Occupation**: College student, part-time barista
**Tech Savvy**: High (grew up with tech)

**Pain Points:**
- Executive dysfunction: hard to start tasks
- Forgets assignments, even important ones
- "Time blindness" — underestimates how long things take
- Overwhelmed by big projects (break them down!)
- Calendar apps feel like homework
- Motivation fluctuates wildly

**Goals:**
- Actually turn things in on time
- Feel less stressed about school
- Build better habits
- Not rely on parents/partner for reminders
- Graduate

**What They Need:**
- AI that breaks big tasks into tiny steps
- Gentle, supportive reminders (not guilt-tripping)
- Time estimation help ("this will take 2 hours")
- Gamification? (streaks, progress)
- Very simple UI (no clutter)
- Quick capture from anywhere

**Will Pay For:** Maybe student pricing, or freemium with core features free

---

### 4. The Busy Parent
**Name**: Maria
**Age**: 38
**Occupation:**
- Part-time nurse
- Mom of 2 (ages 6 and 9)
- PTA volunteer

**Tech Savvy:** Medium

**Pain Points:**
- Family calendar is a nightmare
- School events, doctor appointments, sports practices
- Grocery lists get lost
- "Remind me to..." requests from everyone
- Self-care tasks always get pushed back
- Mental load is exhausting

**Goals:**
- Sync family schedules
- Actually remember appointments
- Delegate tasks to family members
- Find time for herself
- Stop being the family reminder machine

**What They Need:**
- Shared calendars with permissions
- Family task assignments
- Location-based reminders ("at grocery store, pick up milk")
- Voice input (hands-free)
- Very clear, colorful interface
- Family subscription plan

**Will Pay For:** Yes, if the whole family can use it

---

### 5. The Minimalist Power User
**Name**: Riley
**Age**: 26
**Occupation:** Software Engineer
**Tech Savvy:** Very High

**Pain Points:**
- Hates bloated apps
- Uses keyboard shortcuts, hates clicking
- Wants full control
- Syncs across multiple devices
- Automates everything
- Values privacy

**Goals:**
- Fastest possible workflows
- API access for automation
- Data ownership
- Clean, distraction-free UI
- Power-user features

**What They Need:**
- Keyboard shortcuts everywhere
- API / CLI interface
- Customizable views
- Local data option
- Export everything easily
- Dark mode (non-negotiable)

**Will Pay For:** Yes, premium plan with power features

## Secondary Personas

### The Non-Techie Elder
**Name**: Robert, 67, retired teacher
- Needs: Large fonts, clear language, minimal features
- Pain: "Why do apps have so many buttons?"
- Will Pay: Probably not. Free tier only

### The Corporate Team
- **Not MVP**: Collaboration features, team permissions, integrations with Slack/Jira
- **Post-v1**: Build out for this use case

## Prioritization Based on Personas

**MVP (v1) Target**: Alex (Overwhelmed Professional) + Jordan (Freelancer)
- Why? They have budget, clear pain points, and will provide good feedback
- Features: Calendar, tasks, AI suggestions, reminders

**v1.5 Add**: Sam (Student with ADHD)
- Task breakdown
- Gamification elements
- Student pricing tier

**v2 Add**: Maria (Busy Parent)
- Shared calendars
- Family features
- Voice input

**v2+ Consider**: Riley (Power User)
- API access
- Advanced automation
- Privacy-focused local mode

## Anti-Personas (Who We're NOT Building For)

### The Enterprise Project Manager
- Needs: Gantt charts, resource allocation, team workflows
- Our take: Use Monday/Asana/Jira. We're for individuals, not enterprise.

### The Social Scheduler
- Needs: Event invites, RSVPs, social discovery
- Our take: Use Calendly/Meetup. We're about productivity, not social.

### The Habit Tracker Enthusiast
- Needs: Streaks, gamification, social challenges
- Our take: We'll have some habit tracking, but it's not our focus. Use Streaks/Habitica.
